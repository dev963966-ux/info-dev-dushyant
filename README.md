# Free Fire Player Info API

## 🎯 About
Personal project to fetch Free Fire player information using reverse engineered endpoints.

## 🚀 Live Demo
```

https://your-api.vercel.app/player-info?uid=2670799190

```

## 📡 Endpoints

### Get Player Info
```

GET /player-info?uid=PLAYER_UID
GET /player-info?uid=PLAYER_UID&region=IND

```

### Health Check
```

GET /health

```

### API Statistics
```

GET /stats

```

### Refresh Tokens
```

POST /refresh

```

IND, BR, US, SAC, NA, SG, RU, ID, TW, VN, TH, ME, PK, CIS, BD, EUROPE
